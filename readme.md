# Friendlee-test-task 

## Используемые пакеты

- React 17
- Redux (toolkit)
- Typescript 
- Webpack 5
- Babel  
- scss 

## Запуск dev среды 

npm start

## Билд

npm run build

## Конфигурация

- webpack.dev.config.js - dev среда
- webpack.prod.config.js - prod среда
- webpack.test.config.js - test среда

## Команды 

- `npm run lint` Запуск eslint 
- `npm run lint:fix` Запуск фиксов eslint
